“Election Integrity Dashboard MVP”

🎯 OBJETIVO

Crear un sistema que:

Permita subir imágenes de actas electorales
Genere un hash SHA256 único por imagen
Guarde todo en una base de datos real (SQLite persistente)
Cree una cadena tipo blockchain (ledger inmutable encadenado)
Ejecute OCR para extraer números de la imagen
Detecte posibles anomalías simples en el acta
Muestre todo en un dashboard con sidebar minimalista

🗃️ BASE DE DATOS (OBLIGATORIA REAL)

Usar SQLite persistente.

Tabla: actas

Campos:

id (int autoincrement)
mesa_id (string)
image_path (string)
image_hash (SHA256)
ocr_result (text/json)
timestamp
prev_hash
block_hash
🔗 BLOCKCHAIN SIMULADA (LEDGER REAL)

Cada nuevo registro debe encadenarse:

block_hash = SHA256(prev_hash + image_hash + timestamp)

Esto crea una cadena inmutable verificable.
